# 📊 Regional Sales Analysis

A full-stack data analytics project that dives into U.S. regional sales using Python (EDA), Power BI (dashboard), and Excel (data prep). The goal is simple: expose trends, identify performance gaps, and support strategic decision-making.

---

## 🚨 Problem Statement

Sales and profit performance vary drastically across regions and seasons. Without clear insight, sales teams are flying blind.

**Objective:** Use 5 years of historical data to:
- Pinpoint seasonal trends
- Analyze product and customer-level profitability
- Evaluate sales channels and regional performance
- Provide an interactive dashboard for decision-makers

---

## 🧰 Tools & Technologies

- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI
- Excel
